spring-boot-web扩展演示

1. @JsonParam 自动解析json字符串为参数

>>> @JsonParam 用于接收前端的json字符串，并自动转换为Java Bean对象

  • 传递json参数: ./testJsonParam?test={"name":"abc","password":123456,"age":16}

2. 支持返回对象进行下载

>>> 返回DownloadBean和StreamDownloadBean进行下载。

  • 下载文件: ./testDownloadBean
  • 下载文件(stream方式): ./testStreamDownloadBean_1
  • 下载文件(stream方式)-静态文件(smile.jpg): ./testStreamDownloadBean_2

3. 自动解析日期

>>> 支持自动将String类型解析为Date类型，支持多种格式。该方式同时支持@RequestParameter的注入和@JsonParam方式的注入

  • 日期输出: ./testDateToString?time=20180808080808

4. 输出Date类型时，按0时区处理，输出格式yyyy-MM-dd HH:mm:ss

>>> 演示日期的输出，Data, LocalDateTime, LocalDate, LocalTime

  • 日期输出: ./testDateToString?time=20180808
